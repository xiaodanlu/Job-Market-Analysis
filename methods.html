Combine_data.csv: Dataset Overview

The plot 1 to 5 were made by only using the USA dataset. This section only analyzes job openings across the United States. Therefore, adding data specifically from DC dataset would result in biased data rather than random data.

The plot 6 and plot 7 were made by both USA dataset and DC dataset. This section utilizes two datasets as the focus is on job openings in DC. Therefore, the two datasets were reorganized and categorized into DC and the rest of the US. The aim is also to present the audience with all the available job positions in the US up until April 14, 2023.

Data Visualization Methodologies

Name Description Code Dataset
Figure 1: Symbol Map The map was created by using Tableau, a data visualization software. To create the map, the latitude and longitude coordinates for each job posting were obtained using geocoding, which is the process of converting addresses into geographic coordinates. The data was then imported into Tableau, where a symbol map was created using the latitude and longitude coordinates. The size of the dots on the map represented the number of job openings in a given location, while the colors were used to represent different regions of the US. Clean; Tableau Link
Figure 2: Treemap The treemap visualization was created using Tableau. The visualization displays rectangles of different sizes, with each rectangle representing a company and the size of the rectangle indicating the number of job postings available at that company. The larger the rectangle, the more job openings are available at the corresponding company. Clean; Tableau Link
Figure 4: Packaged Bubble Plot

The packaged bubble plot visualization was created using Tableau. The size of each bubble represents the number of job postings available on each site.

Clean; Tableau Link
Figure 5: Box Plot The box plot was made by Plotly. The graph displays the minimum and maximum salary ranges, the median, and the interquartile range (IQR) for each job type. To create the plot, we first filtered the dataset to include only job postings with salary information available and take the average of each job posting. Then we categorized the job postings into different types such as data analyst, machine learning, neural networks, deep learning, reinforcement learning, and natural language processing to analyze the difference in salary between job types. Clean & Plot Link
Plot 6: Word cloud

Word cloud visualization is a technique to analyze and present text data. The font size of each word corresponds to its frequency in the text document.

Clean & Plot Link
Plot 7: Box Plot The box plot is made by using Plotly. The x-axis is the job types included in the visualization are data analyst, natural language processing, blockchain, big data, and neural network. The y-axis is the salaries, that are grouped into two categories: the District of Columbia and surrounding areas, and the rest of the United States. The box plot shows the median, the interquartile range (IQR), and the range of the salaries for each job type and region. Clean 1; Clean 2; Plot Link 1; Link 2
Plot 8: Map with Icon The map visualization is made by using folium and the data for the map was sourced from both the USA and DC databases. The icons on the map are color-coded to represent different job types. When clicked, each icon displays information such as the company name, job title, and website from which the job information was sourced. Clean & Plot Link
